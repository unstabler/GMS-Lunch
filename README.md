GMS-Lunch
=========

모바일고 [오늘의 급식 페이지](http://gms.jubeat.kr) 를 새로 만들었습니다.
이전 버전과의 차이점이라면, 

 * jQuery Mobile로부터의 탈피
 * Perl CGI가 아닌 Dancer 웹 프레임워크의 사용 -> 속도 증가 
 * 트위터 봇과 홈페이지의 모듈 단일화
 * 캐싱 기능 (이전 버전에서는 접속할 때 마다 홈페이지에서 가져왔습니다 ' ');

등이 있겠네요.
